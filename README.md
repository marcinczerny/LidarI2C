# LidarI2C
## Projekt obslug Lidaru Lite v3Hp
### W jezyku Python dla Raspberry Pi

Używa biblioteki `smbus2` do obsługi i2c w języky Python

Zaleca sie uzywanie zestawu gotowych narzedzi na raspbery `i2ctools`.
